Wat is een hash?

Een hash is een manier om een woord om te zetten in een getal of code.
De computer slaat niet het echte wachtwoord op, maar alleen de hash.

Voorbeeld:
hash
H = 8, A = 1, S = 19, H = 8
8 + 1 + 19 + 8 = 36

hash → 36

Wat is binaire code?
Binaire code is een systeem dat alleen de cijfers 0 en 1 gebruikt om informatie weer te geven.
Elke positie in een binaire reeks heeft een waarde die een macht van 2 is (bijvoorbeeld, voor 8-bit: 128, 64, 32, 16, 8, 4, 2, 1 van links naar rechts).
Een '1' op een positie betekent dat de waarde van die positie meetelt (dat die positie dus 'aan staat'), een '0' betekent dat deze niet meetelt (dat die dus 'uit staat').
Bijvoorbeeld, de binaire code `01000001` staat voor: 0*128 + 1*64 + 0*32 + 0*16 + 0*8 + 0*4 + 0*2 + 1*1 = 65.
Computers gebruiken binaire code om gegevens te verwerken en op te slaan.

Elke letter, cijfer of symbool heeft een unieke binaire representatie.

Hier zijn enkele voorbeelden van hoofdletters in 8-bit binaire code:
A: 01000001 (65)
B: 01000010 (66)
C: 01000011 (67)
